# 适用于 ComfyUI 的 Wan Animate 增强器

为 WanAnimateToVideo 提供多维度可控增强

版本：1.0.0

## 功能

- 运动强度控制（身体运动）
- 表情强度控制（面部动画）
- 姿态遵从控制（跟随姿态视频）
- 背景融合控制（角色/背景整合）
- 与原始 WanAnimateToVideo 的参数完全兼容
